# go-salt

## Synopsis

Library for interacting with salts salt-master API.

https://saltstack.com/

## Build status

* [![CircleCI Master](https://circleci.com/gh/r3labs/go-salt/tree/master.svg?style=svg)](https://circleci.com/gh/r3labs/go-salt/tree/master)

## Installing

```
$ make deps
$ make install
```

## Tests

Running the tests:
```
make test
```
